# Sending Email with PHP using PHPMailer

This repository demonstrates how to send emails using PHPMailer, particularly through Gmail. Follow these steps to set it up:

## Step 1: Download PHPMailer Library

1. Visit [PHPMailer GitHub](https://github.com/PHPMailer/PHPMailer).
2. Click on the green "Code" button and download the ZIP file.
3. Extract the contents into your project folder.

## Step 2: Install Composer

1. Download Composer from [getcomposer.org](https://getcomposer.org/) and install it.
2. Open your code editor and run the following command in the terminal: `composer require phpmailer/phpmailer`

## Step 3: Run the `SendEmail-Main` File

Execute the `SendEmail-Main` file provided in this repository.

**Important Note:**
When filling in your information in the code, there's a field labeled "password." This password is your app password, which you can obtain by following these steps:

1. Go to your Google Account.
2. Navigate to "Security."
3. Enable "2-Step Verification."
4. Generate an "App Password."

Google will provide you with a 16-character app password. Copy and paste this password into your code.
